Built with WordPress

This WordPress site uses underscore as a base to build a beautiful parallax site.

Parallax Panes

Each part of the homepage is using a parallax pane,

  • Bio
  • Skills
  • Social
  • Resume
  • Contact

Theme Usage

The Republic is built with underscore, HTML5, CSS3, SASS (compiled locally with codekit), JavaScript, and occasional PHP tweaks to add in everything that the developer wanted to include.
